# SAP CoE - Code Jam Repository!

Welcome to the **Code Jam Repository!**
This repo serves as a central reference hub for everything related to our Code Jam sessions.

## Purpose

The main goal of this repository is to:
- Provide easy access to Code Jam materials - topics covered, workbooks, learning materials etc.
- Maintain a library of hands on sessions and assignments for reference.
- Encourage collaborative learning and improvement.
